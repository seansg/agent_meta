# Agent Meta

This repository contains skills and configuration for the Antigravity Agent.

## Skills

- **adk-tool-scaffold**: Scaffolding for ADK tools.
- **database-schema-validator**: Validates database schemas.
- **git-commit-formatter**: Formats git commit messages (Conventional Commits).
- **git-pr**: Automates Pull Request workflow.
- **git-worktree**: Manages git worktrees.
- **json-to-pydantic**: Converts JSON to Pydantic models.
- **skill-help**: Provides help and documentation for skills.
